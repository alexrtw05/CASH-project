You are alone. 40 light-years from Earth. The Hail Mary is your last hope.

Mission control's final message was not sent in words. It was written in the stars themselves. They ranked every visible star by how high it stood in the sky at a specific moment. They marked the top ones red. The brightness of the red is the shift. The rank order is the arrangement.

Find the red stars. Measure their glow. Undo the shifts. Go home.
// THE FULL PIPELINE
  • Use ephem to compute the altitude of all 15 named stars on the given date from Zurich
  • Sort all 15 stars by altitude descending — highest in sky first
  • Take the top N stars (N = length of the encoded word) — these are the message stars in order
  • For each message star, compute its pixel position using the formula below and read the red channel from the image: img[y, x, 2]
  • For each letter i: decoded[i] = (encoded[i] - red_channel[i] % 26) % 26 — reverse the Caesar shift
  • No transposition to undo — the star altitude order IS the word order
// AZ/ALT TO PIXEL FORMULA

This formula is the same across all levels. Given a star's azimuth and altitude in degrees, compute which pixel in the 800x800 image it occupies.

x = int((az_deg % 360) / 360 * 800) % 800
y = int((90 - alt_deg) / 180 * 800) % 800
Altitude can be negative (star below horizon). Python's % handles this correctly — it always returns a non-negative result unlike some other languages.
// FINDING THE RED PIXELS

Real message stars have B==0, G==0, R >= 28. Decoy red pixels have R < 28. Filter precisely.

# You don't actually need to find them by color —
# you KNOW where they are from the ephem computation.
# Just compute the position and read that pixel directly.

for name, az_deg, alt_deg in top_stars:
  px = int((az_deg % 360) / 360 * 800) % 800
  py = int((90 - alt_deg) / 180 * 800) % 800
  red = int(img[py, px, 2])
  red_channels.append(red)
Note: img[y, x] — row first (y), then column (x). It's the opposite of what you might expect.
